# First Project

A simple Node.js project to practice backend development and GitHub workflows.

---

## Features
- Backend built with **Node.js** and **JavaScript**
- Basic server setup (`index.js`)
- Ready for expansion (Express.js, MySQL, APIs)

---

## Prerequisites
- Node.js
- npm
  
---

## Installation

1. Clone the repository:
```
   git clone https://github.com/BaranBattal/first-project.git
```
2. Navigate into the project directory:
```
   cd first-project
```
3. Install dependencies:
```
   npm install
```
4. Start the server:
```
   node index.js
```
